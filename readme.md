## Bagian 4: State Management 

<img width="658" alt="Screenshot 2023-12-18 075754" src="https://github.com/Fauzan-121140009/UAS_PEMWEB_121140009.github.io/assets/125173289/e0bc5104-5df9-47b2-a8f2-b0aaec635ef9">

Code pada Web ini sudah menggunakan Session_start() . Fungsi session_start() digunakan dalam PHP untuk memulai atau melanjutkan sesi yang ada. Sesi memungkinkan Anda menyimpan data di antara permintaan halaman web, memungkinkan Anda untuk mengidentifikasi pengguna atau menyimpan informasi yang diperlukan di sepanjang interaksi mereka dengan situs web Anda.

Ini adalah fungsi yang penting untuk digunakan di awal skrip PHP jika Anda berencana untuk menggunakan variabel sesi atau melacak informasi pengguna yang spesifik antara berbagai halaman web.

<img width="451" alt="Screenshot 2023-12-18 080011" src="https://github.com/Fauzan-121140009/UAS_PEMWEB_121140009.github.io/assets/125173289/fc6794c9-6c92-4180-9c94-de0dea80a50e">

Code pad web juga telah dilengkapi dengan cookie, Cookie pada web ini dapat digunakan untuk menyimpan informasi otentikasi, membiarkan pengguna tetap masuk ke situs web tanpa perlu login setiap kali mereka mengunjungi halaman baru.
